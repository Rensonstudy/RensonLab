# 🎯 Metas del Proyecto

> "Llegar a la Universidad de los Andes con una preparación superior al promedio en Ingeniería de Sistemas."

---

# Objetivo Principal

Entrar a Ingeniería de Sistemas en la Universidad de los Andes con una base sólida en programación, matemáticas, inglés y pensamiento computacional para aprovechar al máximo la carrera desde el primer semestre.

---

# Objetivos Académicos

## Programación

- [ ] Completar CS50x
- [ ] Dominar Python básico e intermedio
- [ ] Aprender Programación Orientada a Objetos
- [ ] Resolver al menos 200 ejercicios de programación
- [ ] Construir 20 proyectos personales
- [ ] Crear un portafolio profesional en GitHub

---

## Matemáticas

- [ ] Dominar Álgebra
- [ ] Dominar Matemáticas Discretas
- [ ] Reforzar lógica matemática
- [ ] Aprender funciones y conjuntos
- [ ] Desarrollar pensamiento analítico

---

## Linux y Herramientas

- [ ] Manejar Linux con comodidad
- [ ] Aprender Git y GitHub
- [ ] Utilizar VS Code de forma profesional
- [ ] Documentar proyectos correctamente

## Inglés
